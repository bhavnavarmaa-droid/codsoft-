# Data Science Projects

## Overview

This repository contains three data science projects that demonstrate data analysis, preprocessing, feature engineering, predictive modeling, and model evaluation using Python and Scikit-Learn. These projects cover both classification and regression problems using real-world datasets.

---

# 1. Titanic Survival Prediction

## Project Description

The Titanic Survival Prediction project aims to predict whether a passenger survived the Titanic disaster based on various passenger characteristics such as age, gender, passenger class, fare, and family information.

This project is one of the most popular beginner-level predictive analytics projects and helps in understanding classification techniques and data preprocessing methods.

## Objective

To build a predictive model that determines the survival status of passengers using historical Titanic passenger data.

## Dataset Features

* Passenger ID
* Passenger Class
* Name
* Sex
* Age
* Number of Siblings/Spouses Aboard
* Number of Parents/Children Aboard
* Ticket Number
* Fare
* Cabin
* Embarked Port
* Survival Status

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn

## Methodology

* Data Cleaning
* Missing Value Treatment
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Classification Model Development
* Model Evaluation

## Evaluation Metrics

* Accuracy Score
* Precision
* Recall
* F1 Score
* Confusion Matrix

## Outcome

The model successfully predicts passenger survival and provides insights into the factors that influenced survival during the Titanic disaster.

# 2. Movie Rating Prediction

## Project Description

The Movie Rating Prediction project focuses on estimating movie ratings using features such as genre, directors, and actors. By analyzing historical movie data, the project identifies relationships between movie attributes and audience ratings.

## Objective

To develop a regression model capable of predicting IMDb movie ratings based on available movie information.

## Dataset Features

* Movie Name
* Release Year
* Duration
* Genre
* Rating
* Votes
* Director
* Actor 1
* Actor 2
* Actor 3

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Matplotlib
* Seaborn

## Methodology

* Data Preprocessing
* Missing Value Handling
* Feature Engineering
* Text Feature Extraction
* Regression Model Development
* Performance Evaluation

## Evaluation Metrics

* Mean Absolute Error (MAE)
* Root Mean Squared Error (RMSE)
* R² Score

## Outcome

The model learns patterns from historical movie data and generates rating predictions based on genres, directors, and cast information.

# 3. Iris Flower Classification

## Project Description

The Iris Flower Classification project predicts the species of Iris flowers based on their sepal and petal measurements. It is one of the most widely used datasets for introducing classification techniques.

## Objective

To classify Iris flowers into their respective species using flower measurement data.

## Species Classification

* Iris Setosa
* Iris Versicolor
* Iris Virginica

## Dataset Features

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn

## Methodology

* Data Exploration
* Data Visualization
* Feature Analysis
* Classification Model Development
* Model Evaluation

## Evaluation Metrics

* Accuracy Score
* Classification Report
* Confusion Matrix

## Outcome

The model accurately classifies Iris flowers into their respective species based on flower measurements and demonstrates strong classification performance.
